# v0.1.0 首次公开发布

这是项目的首次公开发布版本，目标是提供一个可运行、可理解、可继续扩展的智慧农业管理系统基线。

## 本次发布包含

- 中文管理后台基础框架
- 传感器数据接入、聚合查询与首页运营总览
- 病害识别异步链路与任务状态更新
- `AI 智能解析` 与建议草稿确认入库
- 专家审批、工人执行、结构化完工回填
- 调度中心与可观测中心
- Docker Compose + Nginx 的生产部署骨架

## 适合用来做什么

- 学习智慧农业平台的完整工程化实现
- 作为温室运营管理系统的二次开发基础
- 用于研究“时序数据 + AI 建议 + 任务闭环”的产品设计
- 用作前后端、调度、AI、部署一体化的参考项目

## 发布边界

- 公开仓库不包含真实客户资料、真实参数目录、合作方白皮书或真实密钥
- 真实 Partner API 联调需要你本地自行配置私有参数目录和环境变量
