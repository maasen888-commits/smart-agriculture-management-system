## [0.1.0] - 2026-03-12

### Added

- Initial public repository baseline
- Chinese admin console for greenhouse operations
- Sensor data ingestion, time-series aggregation, and dashboard overview
- Vision upload, async recognition workflow, and task status updates
- AI insights, recommendation drafts, and confirmation into task workflow
- Expert approval and worker execution loop with structured completion report
- Scheduler center and observability center
- Docker Compose + Nginx production deployment skeleton
- Privacy verification scripts for current tree and Git history
- Contributing guide, issue templates, PR template, security policy, release notes, and commercial licensing guidance

### Changed

- Sanitized public templates, example accounts, and example catalog for safe public release
- Switched repository license model to source-available non-commercial distribution

### Security

- Removed private partner documents and sensitive defaults from the public repository baseline
- Introduced repository-level privacy verification workflow before release
